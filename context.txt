1. PROJECTOVERZICHT
---------------------------------------------------------------
Groepsproject van 5 studenten voor VRT (Belgische openbare omroep).
Doel: volledige observability stack (logging + metrics + tracing) bouwen op AWS,
gecorreleerd via trace ID, zodat je een probleem detecteert in < 10 minuten.

Deadline MVP demo: 1 april 2026
Deadline eindoplevering: juni 2026

AWS omgeving: sandbox van VRT, admin-rechten, kostenalarm aanwezig.

---------------------------------------------------------------
3. GENOMEN BESLISSINGEN (samenvatting)
---------------------------------------------------------------
Volledige argumentatie: docs/keuze/toolkeuze.md en docs/keuze/app-keuze.md

Gekozen app: Docker Example Voting App
  - 5 services: Vote (Python) → Redis → Worker (.NET/C#) → PostgreSQL → Result (Node.js)
  - 3-4 hops per request, 3 programmeertalen, geen ingebouwde observability
  - Ligt klaar in /app/ met docker-compose.yml

Gekozen tools (alleen open source, geen SaaS, geen AWS-native):
  - Tracing:  Jaeger + OpenTelemetry Collector
  - Metrics:  Prometheus + Grafana
  - Logging:  Grafana Loki + Promtail
